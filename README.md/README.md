# Gharpayy CRM Backend MVP

**Project Description:**  
Gharpayy CRM Backend MVP is a Minimum Viable Product designed to manage PG accommodation leads in Bangalore.  
It helps capture leads, assign them to agents, track the sales pipeline, and schedule visits efficiently.

---

## **Features Implemented**
- **Lead Capture:** Automatically captures leads from forms or manual entry.
- **Lead Assignment:** Assigns leads to agents automatically (round-robin or workload-based).
- **Lead Pipeline:** Tracks lead status through stages: New → Contacted → Requirement Collected → Property Suggested → Visit Scheduled → Visit Completed → Booked → Lost.
- **Visit Scheduling:** Agents can schedule property visits and mark outcomes.
- **REST APIs:** CRUD APIs available for Leads, Agents, and Visits.
- **Database:** Uses PostgreSQL to store leads, agents, and visits.

---

## **Tech Stack**
- **Backend:** Java, Spring Boot
- **Database:** PostgreSQL
- **Build Tool:** Maven
- **Version Control:** Git & GitHub
- **Testing:** Postman
